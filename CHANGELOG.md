# Changelog

All notable changes to `sniff-cli` will be documented in this file.

## 1.0.0 - 2026-03-19

Initial public release of the renamed and repackaged `sniff-cli` project.

- Renamed the published distribution to `sniff-cli`.
- Renamed the Python import package to `sniff_cli`.
- Renamed the default config file to `.sniff-cli.toml`.
- Standardized the console entrypoint as `sniff-cli`.
- Cleaned up repository metadata, developer docs, and packaging surfaces for reuse.
- Added CI coverage for linting, typing, tests, and package builds.
